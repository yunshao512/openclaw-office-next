# 📸 如何替换头像和全身照

## 快速开始

### 1. 准备您的图片

#### 头像要求
- **尺寸**: 200x200 像素（推荐）
- **格式**: JPG 或 PNG
- **内容**: 清晰的正面头像照片
- **数量**: 每个性别至少 3 张

#### 全身照要求
- **尺寸**: 400x600 像素（推荐，保持比例即可）
- **格式**: JPG 或 PNG
- **内容**: 清晰的全身照片
- **数量**: 每个性别至少 5 张

### 2. 放置图片

#### 女性头像
```
public/avatars/female/face/
├── 1.jpg  ← 替换为您的第1张女性头像
├── 2.jpg  ← 替换为您的第2张女性头像
└── 3.jpg  ← 替换为您的第3张女性头像
```

#### 女性全身照

```
├── 3.jpg  ← 替换为您的第3张女性全身照
```

#### 男性全身照

```
public/avatars/male/fullbody/
├── 1.jpg  ← 替换为您的第1张男性全身照
├── 2.jpg  ← 替换为您的第2张男性全身照

├── 4.jpg  ← 替换为您的第4张男性全身照
└── 5.jpg  ← 替换为您的第5张男性全身照
```

### 3. 命名规则

**重要**: 文件名必须使用数字命名：
- ✅ 正确: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`
- ❌ 错误: `photo1.jpg`, `image.jpg`, `刘亦菲.jpg`

### 4. 添加更多图片（可选）

如果您想添加更多图片，只需继续编号：
- 头像: 可以添加 `4.jpg`, `5.jpg` 等
- 全身照: 可以添加 `6.jpg`, `7.jpg` 等

**注意**: 需要修改代码中的 `maxImages` 参数：
```typescript
// 在 src/components/agent/AgentDetailModal.tsx 中
const maxImages = viewMode === 'face' ? 3 : 5  // 改为您想要的数量
```

## 使用说明

### 在界面中切换图片

1. **点击 Agent 头像** 打开详情窗口
2. **切换模式**：
   - 点击 "头像" 按钮查看头像
   - 点击 "全身" 按钮查看全身照
3. **浏览图片**：
   - 点击左箭头 ◀ 查看上一张
   - 点击右箭头 ▶ 查看下一张
4. **查看页码**：底部显示当前图片编号（如 "2 / 5"）

## 图片建议

### 最佳实践
1. **统一风格**: 保持所有图片风格一致
2. **高质量**: 使用高分辨率、清晰的照片
3. **正面照片**: 头像建议使用正面照
4. **完整全身**: 全身照建议包含完整的身体

### 推荐来源
- 您自己拍摄的照片
- 有版权授权的照片
- AI 生成的虚拟人物照片
- 免费商用图库（Unsplash, Pexels 等）

## 常见问题

### Q: 图片不显示怎么办？
A: 
1. 检查文件名是否正确（必须是数字）
2. 检查文件格式（JPG 或 PNG）
3. 刷新浏览器
4. 检查浏览器控制台是否有错误

### Q: 可以使用 GIF 吗？
A: 不建议，可能会导致性能问题

### Q: 图片尺寸不对会怎样？
A: 系统会自动裁剪适应，但建议使用推荐尺寸以获得最佳效果

### Q: 可以混合使用 JPG 和 PNG 吗？
A: 可以，只要文件名正确即可

## 🎨 图片裁剪和调整功能

### 功能说明

系统支持对图片进行缩放、拖拽和裁剪，让您可以选择图片的最佳展示区域。

### 使用步骤

#### 1. 打开图片设置
- 点击任意 Agent 头像打开详情窗口
- 点击右上角的 **"设置形象"** 按钮

#### 2. 选择图片模式
- **头像模式**: 选择 1:1 比例的展示区域
- **全身照模式**: 选择 2:3 比例的展示区域

#### 3. 调整图片

##### 放大/缩小
- 点击 **放大按钮** (🔍+) 放大图片
- 点击 **缩小按钮** (🔍-) 缩小图片
- 当前缩放比例显示在中间（如 120%）

##### 拖拽移动
- **当缩放比例 > 100% 时**，可以拖拽图片
- 按住鼠标左键并拖动，调整图片位置
- 松开鼠标完成移动

##### 裁剪区域
- 调整裁剪框的大小和位置
- 框内的区域将作为最终显示内容
- 头像裁剪框为正方形
- 全身照裁剪框为竖长方形

#### 4. 切换图片
- 点击 **左箭头** ◀ 查看上一张图片
- 点击 **右箭头** ▶ 查看下一张图片
- 页码显示在底部（如 2 / 5）

#### 5. 保存设置
- 点击 **"确认选择"** 按钮保存
- 点击 **"取消"** 按钮放弃修改

### 重置功能

点击 **重置按钮** (🔄) 可以：
- 恢复默认缩放比例（100%）
- 恢复默认位置（居中）
- 恢复默认裁剪区域

### 注意事项

1. **缩放限制**: 缩放范围 50% - 300%
2. **拖拽限制**: 仅在缩放 > 100% 时可用
3. **自动保存**: 裁剪设置会自动应用到当前图片
4. **独立设置**: 每张图片的裁剪设置独立保存

### 示例场景

#### 场景1: 高清照片需要裁剪头部
```
1. 打开设置 → 选择"头像"模式
2. 放大到 150%
3. 拖动图片使头部居中
4. 调整裁剪框包含整个头部
5. 点击确认
```

#### 场景2: 全身照太大需要缩放
```
1. 打开设置 → 选择"全身"模式
2. 缩小到 80%
3. 调整裁剪框包含完整身体
4. 点击确认
```

## 💡 高级技巧

### 技巧1: 批量处理
如果您有多张图片需要统一风格：
1. 先设置好第一张图片的裁剪
2. 记住缩放比例和位置
3. 对其他图片应用相同设置

- 切换不同图片查看效果
- 随时调整直到满意
- 确认前可以预览最终效果

### 技巧3: 保持一致性
- 头像建议使用相同的缩放比例
- 全身照建议保持统一的裁剪高度
- 这样整个团队看起来更协调

## ❓ 常见问题

### Q: 为什么拖拽不动？
**A**: 拖拽功能仅在缩放比例 > 100% 时可用，请先放大图片。

### Q: 裁剪设置会保存多久？
**A**: 裁剪设置保存在浏览器中，刷新页面后仍然有效。

### Q: 可以撤销裁剪吗？
**A**: 可以！点击重置按钮或重新设置即可。

### Q: 图片显示不完整怎么办？
**A**: 缩小图片或调整裁剪框，确保重要内容在框内。

### Q: 如何查看原始图片？
**A**: 将缩放设置为 100%，裁剪框拉到最大即可。
