# 头像和全身照存储说明

## 目录结构

```
public/avatars/
├── female/              # 女性头像
│   ├── face/           # 头像（200x200 像素）
│   │   ├── 1.jpg
│   │   ├── 2.jpg
│   │   └── ...
│   └── fullbody/       # 全身照（400x600 像素）
│       ├── 1.jpg
│       ├── 2.jpg
│       └── ...
├── male/               # 男性头像
│   ├── face/           # 头像（200x200 像素）
│   │   ├── 1.jpg
│   │   ├── 2.jpg
│   │   └── ...
│   └── fullbody/       # 全身照（400x600 像素）
│       ├── 1.jpg
│       ├── 2.jpg
│       └── ...
└── examples/          # 示例图片（请勿删除）
```

## 图片要求

### 头像（face）
- 尺寸：200x200 像素
- 格式：JPG 或 PNG
- 内容：清晰的正面头像
- 文件命名：1.jpg, 2.jpg, 3.jpg ...（数字顺序）

### 全身照（fullbody）
- 尺寸：400x600 像素（或等比例）
- 格式：JPG 或 PNG
- 内容：清晰的全身照
- 文件命名：1.jpg, 2.jpg, 3.jpg ...（数字顺序）

## 最少数量要求
- 每个性别至少 3 个头像
- 每个性别至少 5 个全身照

## 使用说明
1. 将您的图片放入对应目录
2. 确保文件名为数字（1.jpg, 2.jpg, 3.jpg...）
3. 系统会自动根据 Agent ID 分配图片
4. 点击头像可以切换查看不同照片
