# 监控区自动滚动功能说明

## 更新日期
2026-03-09 17:38

## 新增功能

### 1. 自动循环滚动展示
- **效果**: 日志自动向上滚动，类似跑马灯效果
- **循环**: 日志列表会复制一份，实现无缝循环滚动
- **速度**: 30 秒滚动一轮
- **交互**: 鼠标悬停时暂停滚动，方便查看

### 2. 自动数据刷新
- **刷新间隔**: 从 10 分钟缩短为 30 秒
- **自动更新**: 无需手动刷新页面
- **状态提示**: 显示"更新中..."提示和最后更新时间

## 实现原理

### CSS 动画滚动
```css
@keyframes scrollUp {
  0% {
    transform: translateY(0);
  }
  100% {
    transform: translateY(-50%);
  }
}

.animate-scroll-up {
  animation: scrollUp 30s linear infinite;
}

.animate-scroll-up:hover {
  animation-play-state: paused;
}
```

### 关键技术点

1. **无缝循环**: 将日志数组复制一份 `[...logs, ...logs]`
2. **50% 偏移**: 滚动到 -50% 时正好是复制的日志开始位置
3. **动画暂停**: hover 时暂停，方便用户查看
4. **自动刷新**: 每 30 秒调用一次 API 更新数据

## 使用体验

### 正常状态
- 日志从底部向上缓慢滚动
- 最新的日志在最上面
- 30 秒滚动一轮后重新开始

### 鼠标悬停
- 滚动暂停
- 可以仔细查看当前日志内容
- 移开鼠标后继续滚动

### 数据更新
- 每 30 秒自动获取最新日志
- 显示"更新中..."提示
- 更新后继续滚动展示

## 性能优化

1. **CSS 动画**: 使用 transform 而不是 top，利用 GPU 加速
2. **虚拟滚动**: 只渲染可见区域的日志
3. **智能刷新**: 只在有新数据时更新状态
4. **内存管理**: 最多显示 100 条日志

## 配置说明

```typescript
// 轮询间隔（毫秒）
pollInterval = 30000  // 30秒

// 滚动动画时长（秒）
animation: scrollUp 30s linear infinite
```
